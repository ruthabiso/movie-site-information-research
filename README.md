# movie-site-information-research
本仓库用于对互联网上已公开存在的影视类网站进行系统化信息研究整理，重点围绕电影与电视剧相关站点的基础公开资料进行归纳与汇编。通过统一整理方式，对站点名称、访问地址、站点定位说明及基础分类信息进行结构化记录，便于长期保存与反复查阅。本项目不涉及任何影视资源的存储、播放、下载或解析行为，仅针对公开网络环境中可获取的信息进行研究性整理。所有内容均来源于公开渠道，整理过程保持客观、中立，不对站点内容质量、合法性或服务稳定性作出任何判断或评价，仅作为信息研究与资料参考用途存在。
